Annapurna SIL - Font Features

Annapurna SIL is an OpenType-enabled font family that supports the Devanagari script. It includes a number of optional features that may be useful or required for particular uses or languages. These OpenType features are primarily specified using four-letter tags (e.g. 'cv04').

This document lists all the available features. For more information on how to access OpenType features in specific environments and applications, see Using Font Features.

This page uses web fonts (WOFF2) to demonstrate font features and should display correctly in all modern browsers. For a more concise example of how to use Annapurna SIL as a web font see Annapurna SIL Webfont Example. For detailed information see Using SIL Fonts on Web Pages.

If this document is not displaying correctly a PDF version is also provided in the documentation/pdf folder of the release package.

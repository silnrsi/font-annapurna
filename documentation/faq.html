Here are a few of the most frequently asked questions specifically regarding Annapurna SIL:

My language uses variant forms of some Devanagari characters. How do I type these using the Annapurna font?

If your application supports Graphite, the OpenType Stylistic Sets or Character Variants features, you can use these to access the font features built into the font. See Font features for more details.

What characters are included with this release?

Annapurna SIL covers all the characters from the Devanagari, Devanagari Extended and North Indic Number Forms blocks in The Unicode Standard plus the Rupee currency symbols. See the Design and Type Sample page to see the design of these characters. For the full listing of characters in Annapurna, see the Character set support page.
